# ChrY-Gene-Selection

## Method

Input data: 
- `zimmer_chrY.vcf` with 2,998 germline SNPs on chromosome Y, all passing GATK VQSR quality filtering (FILTER = PASS). 
- Genome build: GRCh37
Gene annotations: 
Retrieved all 542 chrY gene records from the Ensembl GRCh37 (Reference [1]) in 5 Mb chunks
covering all biotypes (protein-coding, lncRNA, pseudogene, etc.)
Coordinates are 1-based, matching VCF format directly.
Variant-gene intersection: 
For each variant position, we identified all gene bodies (start ≤ POS ≤ end), it overlaps using a pandas IntervalIndex. 
498 of 2,998 variants (16.6%) overlapped at least one gene across 117 genes. 
The remaining 2,500 variants (83.4%) are intergenic — expected for chrY, which has large heterochromatic and ampliconic repeat regions with no annotated genes.
Mutational burden ranking: 
Genes were ranked by raw variant count (number of overlapping SNPs).

## Key Notes & Limitations
Raw count is length-dependent: PCDH11Y leads partly because it is the longest gene (742 kb). This is the correct metric per the task definition, but worth noting for interpretation.
Biotype mix: 4 protein-coding genes, 2 lncRNAs, and 4 pseudogenes appear in the top 10 — reflecting chrY's gene composition in GRCh37.

All variants are homozygous alternate (AF=1, AC=2, AN=2 in the VCF INFO) — consistent with a hemizygous Y chromosome in a male individual.

GRCh37 annotation is older: Some genes present in GRCh38 (e.g., newer Y-linked gene models) are absent from this annotation. The analysis is internally consistent with the VCF's reference build.

# ChrY-Gene-Selection
**Sample:** Carl Zimmer Personal Genome (publicly available)  
**Chromosome:** Y  
**Task:** Part 1A — Top 10 Genes by Mutational Burden  
**Reference Build:** GRCh37 (hs37d5)  
**Tools:** bcftools v1.16 + SnpEff v4.3t (GRCh37.75)  
**Date:** 2026-04-05

## Method

### Input data
- `zimmer_chrY.vcf` with 2,998 germline SNPs on chromosome Y, all passing GATK VQSR quality filtering (VQSR (GATK ApplyRecalibration v2.8), SNP mode, ts_filter_level = 99.0). 
  
### Database

- Genome Reference Panel Database:  `hs37d5 (GRCh37/hg19)`
- Gene Models: `Ensembl Release 75 (GRCh37.75)`. This legacy version is required for 100% compatibility with the GRCh37 reference build.

### Software

- VCF Quality Control: `bcftools (v1.16)`
- Functional Annotation: `SnpEff (v4.3t)`
- `Python == 3.11` with `pandas, Matplotlib, Seaborn`

## Analytical Workflow

### Step 1: Quality Control (QC)

The Ti/Tv ratio (1.32) is checked against chrY expectations. Our pipeline identifies heterozygous "artifact" calls (18.2%). Since the Y chromosome is hemizygous in males, these are correctly attributed to mapping errors in highly repetitive regions (e.g., palindromes and X-transposed regions) rather than true variants.

### Step 2: Functional Annotation

Each of the 2,998 variants is processed to determine its location, all variants are classified as strictly genic, upstream/downstream, or intergenic. Noting that only 3 missense variants exist across the entire chromosome, which is consistent with strong purifying selection on the Y chromosome.

### Step 3: Gene Prioritization

Only variants falling strictly within gene bodies (introns, exons, UTRs) are counted toward the burden.

Genes are ranked based on raw count. Top-ranked genes like `PCDH11Y` explains that its rank is a combination of its exceptional length and its location in the X-transposed region.
